Project Overview

This portfolio focuses on learning the fundamentals of MySQL by working with a multi-level company database. It includes writing SQL queries, designing table structures, creating EER diagrams, and executing SQL scripts that define the database and relationships.

Download Document

Database Structure

  • Employees table includes auto-incrementing ID, name (not null), and manager reference.
  • Departments table includes auto-incrementing ID and department name.
  • Employee_Departments uses composite keys to map employees to departments.
  • Employee_Projects stores project names per employee (not null).
  • Managers table relates managers to employees using foreign key references.
